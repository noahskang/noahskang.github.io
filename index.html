Transitninja

Transitninja is an iOS application that allows users to get public transit directions and track bus locations in real-time. It uses the 511.org API to render bus locations on a map. I built this with a team in which I principally served as the Front-End Engineer.

Created with MongoDB, Express, React Native, and NodeJS
